# shortening url
  URL을 입력받아 짧게 줄여주고, Shortening된 URL을 입력하면 원래 URL로 리다이렉트하는 URL Shortening Service

## 프로젝트 환경

* Java 11
* Gradle 6.8.3
* Spring Boot 2.5.6

## 기술

- Springboot
- h2
- MyBatis
- React

## 빌드 및 실행 방법

```
1. ./gradlew build
2. cd build/libs
3. java -jar shortening-url-0.0.1-SNAPSHOT.jar
4. http://localhost:8080에 접속
```
